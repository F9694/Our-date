一个**专属约会邀请网页**：对方打开链接，选择约会项目、日期和时间，确认后再同步日历或把结果发回来。

### 已实现的功能

- 专属姓名和开场文案
- 约会项目选择：咖啡、吃饭、电影、散步、逛展等
- 自定义约会内容
- 推荐日期和自定义日期
- 推荐时间和自定义时间
- 填写饮食、接送等小要求
- 生成专属约会确认卡
- 爱心和彩纸确认动画
- 微信/手机系统分享
- 自动生成 `.ics` 日历文件
- 手机、电脑自适应
- 不需要安装 App，也不需要服务器

### 修改成你自己的内容

解压后打开 `config.js`，修改最前面的内容：

```javascript
window.DATE_APP_CONFIG = {
  fromName: "a",
  toName: "她的名字",
  openingText: "不是群发，也不是临时起意。只是认真地，想把一段时间留给你。",
};
```

活动内容也在这个文件中，例如：

```javascript
{
  id: "coffee",
  icon: "☕",
  title: "喝杯咖啡",
  subtitle: "慢慢聊，不赶时间"
}
```

可以改成：

```javascript
{
  id: "hotpot",
  icon: "🍲",
  title: "一起吃火锅",
  subtitle: "你负责选锅底，我负责订位置"
}
```
